# 📊 Unemployment Rate Analysis (COVID-19 Impact)

This project analyzes unemployment trends over time and studies the impact of COVID-19 on unemployment rates using data visualization techniques.

---

## 📌 Project Overview
The project performs:
- Data cleaning and preprocessing
- Trend analysis of unemployment rate
- Distribution analysis
- COVID-19 impact comparison
- Seasonal (monthly) trend analysis

---

## 🧠 Key Features
- 📈 Time series visualization
- 📊 Distribution plots
- 🦠 COVID-19 impact analysis
- 📅 Monthly trend analysis

---

## 📂 Dataset
- File: `Unemployment.csv`
- Important Columns:
  - Date
  - Estimated Unemployment Rate (%)
